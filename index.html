Setting up Power Automate

  1. 1 Go to flow.microsoft.com and sign in with the admin mailbox account.
  2. 2 Click + Create → Instant cloud flow. Name it something like "Calendar Note from Portal".
  3. 3 Choose "When an HTTP request is received" as the trigger. Click Create.
  4. 4 In the trigger, paste this into Request Body JSON Schema:
    {"type":"object","properties":{"title":{"type":"string"},"body":{"type":"string"},"start":{"type":"string"},"end":{"type":"string"}}}
  5. 5 Click + New step → search for Office 365 Outlook → choose "Create event (V4)".
  6. 6 Set the calendar to the admin shared mailbox calendar. Map: Subject → title, Body → body, Start time → start, End time → end from the trigger.
  7. 7 Save the flow. Go back to the trigger step — copy the HTTP POST URL and paste it here.
